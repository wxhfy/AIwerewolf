# MBTI Persona 头像生成提示词（image2）

> 用途：为数据库中的 37 个 persona 生成统一风格头像。
> 注意：这些是“玩家人格 / MBTI 角色”，不是狼人杀职业身份牌。不要把他们固定画成狼人、预言家、女巫、守卫等职业。

## 统一风格

建议所有头像统一使用以下方向：

- 画幅：1:1 方形头像，胸像或半身像，人物居中，头肩完整，边缘留白。
- 美术风格：现代东方悬疑桌游角色头像，半写实 2D 数字插画，电影感打光，暗夜村庄 / 桌游推理氛围。
- 背景：低对比、虚化、深色暖光，不出现文字、UI、身份牌、职业图标。
- 表情：突出人格特征和推理状态，不要夸张动漫 Q 版。
- 服饰：现代现实服装，可带轻微戏剧化元素，但不要奇幻盔甲化。
- 生成限制：不要文字、不要水印、不要 logo、不要多人物、不要畸形手指、不要过度暴露、不要血腥。

通用反向提示词：

```text
文字，水印，logo，签名，多人物，低清晰度，畸形脸，畸形手，额外手指，过度动漫化，Q版，儿童形象，血腥，恐怖怪物，狼人变身，明确职业身份牌，预言家水晶球，女巫尖帽，猎人枪械特写，守卫盾牌特写，背景过亮，强烈赛博朋克
```
