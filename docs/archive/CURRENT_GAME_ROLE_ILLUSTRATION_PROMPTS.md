# 当前对局职业插画提示词（image2）

> 当前最新 7 人对局职业集合：狼人、女巫、猎人、预言家、村民、守卫。
> 用途：生成职业身份牌插画 / 职业展示图，不是 MBTI 人格头像。

## 统一风格

- 画幅：竖版职业卡插画，建议 3:4 或 2:3。
- 风格：现代东方暗夜狼人杀桌游，半写实 2D 数字插画，电影感光影。
- 构图：单个职业主体居中，半身到全身均可，背景有职业氛围但不要复杂到抢主体。
- UI 适配：人物和关键道具清晰，边缘留白，适合作为游戏职业展示图。
- 不要在图里写职业名称；职业名由前端 UI 显示。

通用反向提示词：

```text
文字，水印，logo，签名，多人物，低清晰度，畸形脸，畸形手，额外手指，过度动漫化，Q版，血腥，恐怖怪物，廉价卡牌边框，复杂 UI 元素，职业名称文字，现代枪械广告感，过度性感，背景过亮
```
