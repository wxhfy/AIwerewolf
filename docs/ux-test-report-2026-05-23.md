## 二、发现的新问题

### ❌ 问题1: 人机模式 ActionPanel 不渲染

**现象**: 人机模式直接访问游戏页时，pending_input 存在但 ActionPanel 不显示。页面显示"已就绪"+"点击运行一局开始观战"，玩家卡片也是占位符。

**原因**: 游戏页的 `startHumanGame()` 需要手动点击按钮触发，没有像 AI 模式那样的自动启动逻辑。从大厅弹窗流程正常（pre-start了游戏），但直接 URL 访问或刷新后会丢失 gameState。

**建议**: 人机模式也加自动启动，或 `useEffect` 检测 `mode===human && !gameState` 时自动调用 `startHumanGame()`。

### ⚠️ 问题2: 玩家卡片不显示自己角色

**现象**: 人机模式 seat=5 的卡片显示"身份隐藏"，应先显示自己的角色和狼队友。

**原因**: `showOwnRole` prop 依赖 `isHumanMode && p.seat === humanSeat && viewMode !== "moderator"`。游戏页默认 `viewMode` 可能是 "moderator"（AI 模式的 auto-set 逻辑）。

**建议**: 人机模式下无论 viewMode 如何都显示自己的角色。
